jibiki REST Application Programming Interface

Summary

command result
GET api/ returns the list of resources stored on the server
GET api/DictName returns the dictionary metadata concerning DictName
PUT api/DictName updates the dictionary metadata concerning DictName
POST api/DictName creates the dictionary DictName
GET api/DictName/lang returns the lang volume metadata concerning DictName
PUT api/DictName/lang updates the lang volume metadata concerning DictName
POST api/DictName/lang creates a new lang volume for DictName
GET api/DictName/lang/prefix returns a list of entry ids whith a headword that match the prefix
GET api/DictName/lang/#id returns the entry with the corresponding id
PUT api/DictName/lang/#id updates the entry with the corresponding id
POST api/DictName/lang/headword creates a new entry with headword in the lang volume
  • All the answers are in XML and encoded in UTF-8.
  • lang is the ISO 639-2/T 3 letter code of the language (eg: 'eng' for English,'esp' for Spanish,'fra' for French, etc.)

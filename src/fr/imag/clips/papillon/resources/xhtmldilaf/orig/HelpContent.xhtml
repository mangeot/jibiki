Aide

Recherche

Pour rechercher un mot dans le dictionnaire, utiliser le menu de recherche gauche comme suit:

  • Etape 1: entrer le mot à rechercher;
  • Etape 2: spécifier la langue en question;
  • Etape 3: choisir la langue destination pour obtenir la traduction du mot;
  • Etape 4: cliquer sur le bouton "go".

Notez que certains dictionnaires ont très peu d'entrées disponibles. Vous n'obtiendrez pas la traduction dans toutes les langues pour tous les mots. La présente interface est expérimentale.

Navigation

Vous trouverez ci-dessous la liste des logiciels recommandés pour optimiser la navigation sur ce site.

Logiciels qui fonctionnent bien

  • Chimera
  • Mozilla 1.0
  • Netscape navigator 7.0
  • OmniWeb 4.1
  • Opera
  • Safari

Logiciels susceptibles de problèmes

  • Amaya: La dernière version traite seulement des caractères xxxxx et non yyyyy.
  • iCab: l'onglet "option" des formulaires (HTML forms) n'est pas bien implémenté.
  • Internet Explorer: l'onglet "option" des formulaires (HTML forms) n'est pas bien implémenté. Il n'affiche pas correctement les caractères non-ascii.

Edition en ligne

Edition d'une entrée monolangue (Lexème)

Vous devrez être enregistré et logged pour éditer des entrées en ligne.

L'interface d'édition en ligne est expérimentale.

Vous pourrez ajouter des entrées via Interface de consultation. Cependant, vos contributions ne seront pas intégrées dans la base de données.

Edition d'un lien de traduction (Axe)

Vous devrez être enregistré et logged pour éditer des liens.

Cette Interface d'édition en ligne est expérimentale.

Cependant, vos contributions ne seront pas intégrées dans la base de données.

Edition hors ligne

Vous pourrez écrire des entrées localement sur votre ordinateur et ensuite les charger sur la base de données.

Pour écrire des entées, nous recommandons l'outil XML Spy pour PC disponible sur www.xmlspy.com pour 30 jours gratuitement, version d'évaluation.

Les schema XML de DiLAF et exemples se trouvent à: www-clips.imag.fr/geta/services/dml/.

Pour tester la validité de vos entrées en ligne, vous pourrez utiliser le W3C XML Schema validator: www.w3.org/XML/Schema.

Gestion des contributions

Vous devrez être enregistré et logged pour gérer vos contributions.

Accédez à Interface d'administration des contributions. Avec cette interface, vous pourrez voir et supprimer vos contributions.

Edition des contributions

Vous devrez être enregistré et logged pour accéder à cette rubrique.

Vous devrez être membre du groupe des spécialistes pour valider une contribution et l'intégrer définitivement dans la base de données.

Vous pourrez éditer les contributions en ligne via l'Interface d'édition des contributions .

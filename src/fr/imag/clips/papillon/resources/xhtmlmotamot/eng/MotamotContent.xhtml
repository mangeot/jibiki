Welcome to the home page of the Motamot project

Here is the list of available bilingual dictionaries:

  • English-French dictionary engfra
  • English-Vietnamese dictionary engvie
  • French-Vietnamese dictionary fravie

The main goal of the project is to participate in the computalization of under-resourced languages. It consists in elaborating a multilingual lexical system by building simultaneously several bilingual dictionaries sharing at least one language between each-other. The construction of these bilingual dictionaries will be made online on a "Papillon" like website built on top of the Jibiki platform. The working methodology is volunteer and collaborative, inspired from the Wikipedia project.

The bilingual link created during the entry editing will be used in order to generate automatically reverse bilingual links and also to create new interlingual links.

The main goals of the project are:

  • launching a contributing process around the construction of each bilingual dictionary of the project. The success of Wikipedia project shows that it is possible, providing that the tools used are simple and easy to use.
  • developing at a large scale small experimental resources like DiCo database (Mel’čuk & Polguère 2006) or PARAX system (Blanc 1996).
  • settling an experimental environment for validating several hypothesis coming from preceding work: bijectivity of bilingual links and transitivity of interlingual ones; massive contribution on the Web; construction of a multilingual lexical system (Polguère 2006).

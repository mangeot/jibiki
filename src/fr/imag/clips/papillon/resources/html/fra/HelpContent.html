Consultation

Pour chercher un mot dans les dictionnaires accessibles sur le serveur Papillon, vous devez utiliser le menu de consultation à gauche de la manière suivante :

  • Étape 1 : entrez le mot recherché;
  • Étape 2 : indiquez la langue du mot entré précédement;
  • Étape 3 : choisissez une langue cible pour obtenir la traduction du mot entré;
  • Étape 4 : appuyez sur le bouton "lancer".

Veuillez noter que certains dictionnaires ne contiennent que très peu d'articles. Le dictionnaire Papillon notamment est en cours de construction. Vous ne trouverez pas de traduction dans la langue désirée dans tous les cas.

Le menu de consultation de gauche proposes une consultation basique. Pour des recherches plus spécifiques comme ne pas tenir compte de la casse, recherche de sous-chaîne, etc., utilisez l'interface de recherche avancée

.

Logiciels problématiques

  • Amaya La dernière version semble afficher les caractéres japonais mais ce n'est pas encore le cas pour le thaï par exemple.
  • iCab ne met pas correctement en œuvre l'élément "option" des formulaires HTML. Il n'affiche pas correctement l'attribut "label" des options. Les caractères non latins ne seront donc pas affichés correctement dans les options. Toutefois, si vous choisissez une langue écrite en caractères latins comme langue préférée de l'interface, vous n'aurez pas ces problèmes.
  • Internet Explorer contient le même problème pour l'attribut "label".

    • Other problems

    Le site Web papillon est entièrement encodé en Unicode UTF-8. Si vous rencontrez des problèmes lors de la visualisation de certaines pages du site, par exemple si vous voyez des points d'interrogation "?" ou des carrés au lieu de caractères chinois ou japonais, vérifiez que l'encodage de votre navigateur est bien Unicode UTF-8.

    si le problème persiste, votre ordinateur peut ne pas avoir de police unicode installée correctement. Essayez d'installer une police Unicode à large couverture comme "Arial Unicode MS".

Édition en ligne

Édition d'un article monolingue (Lexie)

Vous devez être enregistré et logué pour accéder à l'interface d' édition en ligne.

De plus, l'éditeur fonctionne avec des cookies. Vérifiez que votre navigateur les accepte.

L'interface d'édition en ligne est expérimentale. Pour pouvoir implémenter correctement l'éditeur, nous avons développé l'interface la plus complète contenant tous les détails. De ce fait, cette première version implémente une interface pour des lexicographes spécialistes contenant toutes les informations. Grâce à l'éditeur générique et le fait que l'interface complète existe, nous serons en mesure de développer plus facilement des version ssimplifiées de cette première interface en mode dégradé pour des utilisateurs non spécialistes.

Vous pouvez ajouter des articles et ensuite les consulter à travers l'interface de consultation.

Néanmoins, ces contributions risquent de ne pas conservées ni intégrées à la base.

Édition des liens de traduction (Axies)

Vous devez être enregistré et logué pour accéder à l'interface d' édition en ligne.

L'interface d'édition en ligne est expérimentale. Pour pouvoir créer des liens de traduction (axies) entre deux entrées (lexies) de différentes langues, les lexies doivent d'abord être présentes dans le dictionnaire Papillon (et uniquement dans ce dictionnaire). Si elles n'existent pas, vous devez tout d'abord les créer avec l'interface précédente d'édition d'entrées monolingues. Ensuite, une fois que vos deux lexies sont créées, vous pouvez rajouter un lien de traduction (axie) entre elles.

Celle-ci est complètement expérimentale. Vous pouvez ajouter des liens entre articles existants. Néanmoins, ces contributions risquent de ne pas être intégrées à la base.

Édition en local

Vous pouvez rédiger vos lexies localement sur votre ordinateur puis les envoyer pour révision avant intégration dans la base.

Pour la rédaction des lexies, nous conseillons d'utiliser XML Spy pour Windows disponible sur :www.xmlspy.com avec une licence d'évaluation de 30 jours. XMLSpy utilise les schémas XML pour valider les données. Il va télécharger automatiquement les schémas Papillon pour valider vos données.

Les schémas Papillon et des exemples sont disponibles à  www-clips.imag.fr/geta/services/dml/.

Pour tester la validité de vos lexies en ligne, vous pouvez aussi utiliser le validateur de schémas XML du W3C www.w3.org/XML/Schema.

Contributions

Gestion des contributions

Vous devez être enregistré et logué pour gérer vos contributions en ligne en accédant à l' interface de gestion des contributions. Vous pouvez voir ou supprimer des contributions.

Si vous êtes dans le groupe des spécialiste réviseurs, vous pouvez réviser et valider une contribution, c'est-à-dire l'intégrer à la base de données.

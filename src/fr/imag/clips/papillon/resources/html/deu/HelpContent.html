Abfrage

Um ein Wort in den verfügbaren Wörterbüchern des Servers nachzuschlagen, benutzen sie das „Abfrage-Interface“ auf folgende Weise.

  • 1. Schritt: Geben Sie das gesuchte Wort ein
  • 2. Schritt: Geben Sie die Sprache des eingegebenen Wortes ein
  • 3. Schritt: Wählen Sie null, eine oder mehrere Zielsprachen um die Übersetzung des eingegebenen Wortes zu erhalten
  • 4. Schritt: Klicken Sie die Taste „Lookup“

Beachten Sie bitte, dass das Papillon-Wörterbuch eigentlich noch nicht existiert. Die meisten verfügbaren Wörterbücher enthalten also nur ein paar Einträge. Das aktuelle Interface wurde vor allem zu experimentellen Zwecken geschaffen.

Navigation

Um sich optimal durch die Site zu bewegen, geben wir im Folgenden an, welche Software geeignet ist:

Software die ordentlich funktioniert

  • Chimera
  • Mozilla 1.0
  • Netscape navigator 7.0
  • OmniWeb 4.1
  • Opera
  • Safari

Software die Probleme aufwirft

  • Internet Explorer: Das „Option“-Element der HTML-Formulare ist nicht korrekt implementiert. Es zeigt die Option-Kennung anstatt des Option-Textes an. Daher werden nicht-lateinische Schriftzeichen in den Option-Feldern nicht richtig angezeigt. Beachten Sie, dass dieses Problem nicht auftauchen sollte, wenn Sie als bevorzugte Interface-Sprache eine Sprache in lateinschen Buchstaben wählen.
  • iCab: Dieses Programm hat dieselben Problem wie Internet Explorer.
  • Amaya: Dieses Programm kann noch nicht mit nicht-lateinschen Schriftzeichen umgehen.

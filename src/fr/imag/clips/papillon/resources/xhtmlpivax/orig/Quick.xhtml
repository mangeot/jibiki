PIVAX infinitive loop search interface

Send to PIVAX (Quick.po) with following parameters by PROGRAM. It sends back a HTML file with result in a pure HTML format in a gradually process, finished by the EOF character.

Your program saves this HTML file in your application site and load immediatly it when you need

The important parameter is:
  • required: source=[3_letters], target=[3_letters], lemma=[true/false], content=[segment content], volume=[volume name, if you know :), if not PIVAX will search all volumes with the giving source language]
  • optional: timeout=xxxx(ms), scoremin=aaaa, scoremax=bbbb
  • specified and send by SECTra_w (or other ?): segmentId=abcd, project=EOLSS, page=num

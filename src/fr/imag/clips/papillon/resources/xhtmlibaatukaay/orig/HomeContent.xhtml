Le projet iBaatukaayuSunugal

En général les langues africaines sont des langues peu dotées d'où un manque criard de documents, de dictionnaires, de littérature sous format électronique. La plus part des ressources qui exitent en général sont sous format papier. Ce pojet dont la finalité est la mise sur pied d'une base lexicale pour les langues nationales du Sénagal se veut une base de départ ou une référence pour les langues nationales du Sénégal car on ne peut pas parler correcteurs orthographiques, de traducteurs automatique sans parler de dictionnaires électrniques. Comme nous le savons, les langues nationales au sénégal comme le wolof gagne de jour en jour du terrain mais pour leur utilisation dans le sysytème académique nécessite un outillage de ces languese. Le projet ibaatukaayuSunugal a pour but de créer une base lexicale multilingue contributive sur le web pour les langues nationales du Sénégal comprenant entre autres le Wolof, le Pulaar, le Bambara, le Seerer,... L’accès est gratuit pourvu que l’usage ne soit pas commercial (licence de logiciel libre). Ce projet se veut utile et ouvert à la collaboration de toutes les personnes ayant un intérêt pour ces langues.
